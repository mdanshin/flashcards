Привет, GitHub Pages!

Этот проект демонстрирует минимальный сайт, который можно опубликовать через GitHub Pages.

Как опубликовать

  1. Создайте новый репозиторий на GitHub и загрузите в него эти файлы.
  2. В настройках репозитория включите GitHub Pages, выбрав ветку main и папку / (root).
  3. Подождите несколько минут — сайт будет доступен по адресу https://<ваш_логин>.github.io/<имя_репозитория>/.
